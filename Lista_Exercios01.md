# Exercícios sobre Variáveis em Python

## Exercício 1 – Criando Variáveis

Crie variáveis para armazenar seu **nome**, **idade** e **cidade**. Depois, exiba essas informações usando `print()`.

**Exemplo de saída esperada:**

```
Nome: Maria
Idade: 25
Cidade: São Paulo
```

---

## Exercício 2 – Operações com Variáveis

Crie duas variáveis numéricas, `a` e `b`.

* Some, subtraia, multiplique e divida essas variáveis.
* Exiba o resultado de cada operação.

**Dica:** Use `print("Soma:", a + b)`, etc.

---

## Exercício 3 – Atualizando Variáveis

Crie uma variável `saldo` com valor inicial de 1000.

* Adicione 250 ao saldo.
* Subtraia 150 do saldo.
* Exiba o valor final do saldo.

---

## Exercício 4 – Variáveis de Texto e Concatenação

Crie duas variáveis de texto: `primeiro_nome` e `sobrenome`.

* Combine essas duas variáveis para formar uma variável `nome_completo`.
* Exiba o `nome_completo`.

---

## Exercício 5 – Tipos de Variáveis

Crie três variáveis:

* Um número inteiro (`idade`)
* Um número decimal (`altura`)
* Uma variável booleana (`tem_carteira`)

Depois, use a função `type()` para mostrar o tipo de cada variável.

**Exemplo de saída esperada:**

```
<class 'int'>
<class 'float'>
<class 'bool'>
```
