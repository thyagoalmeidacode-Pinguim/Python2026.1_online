# Atividade 2 -- Sistema de Estacionamento Inteligente

## Contexto

Um estacionamento precisa controlar as vagas disponíveis e os carros
estacionados.

## Desafio

Crie um sistema que:

-   Represente o estacionamento como uma matriz (lista de listas)\
-   Cada posição representa uma vaga (livre ou ocupada)\
-   Permita:
    -   Estacionar um carro\
    -   Retirar um carro\
    -   Mostrar o estado atual do estacionamento

## Requisitos obrigatórios

-   Utilizar matriz para representar vagas\
-   Usar estrutura de repetição para percorrer o estacionamento\
-   Usar decisões (if) para verificar se a vaga está livre ou ocupada\
-   Trabalhar com listas e variáveis\
-   Criar um menu interativo

## Desafio extra

-   Mostrar quantas vagas estão livres e ocupadas\
-   Permitir escolher a vaga manualmente
