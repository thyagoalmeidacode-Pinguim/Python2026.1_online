# Atividade 1 -- Sistema de Controle de Biblioteca

## Contexto

Uma biblioteca precisa de um sistema simples para organizar seus livros
e controlar empréstimos.

## Desafio

Desenvolva um programa que permita:

-   Cadastrar livros (título, autor, quantidade)\
-   Listar todos os livros\
-   Realizar empréstimo (diminuir quantidade)\
-   Realizar devolução (aumentar quantidade)

## Requisitos obrigatórios

-   Utilizar listas para armazenar os livros\
-   Utilizar tuplas para representar cada livro (ex:
    `(titulo, autor, quantidade)`)\
-   Criar um menu com estrutura de repetição (while)\
-   Usar estrutura de decisão (if/elif) para as opções\
-   Trabalhar com variáveis para controle do sistema

## Desafio extra

-   Mostrar quais livros estão com estoque zerado
