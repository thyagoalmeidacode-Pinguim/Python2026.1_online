# Atividade 3 -- Sistema de Compras (Mini Mercado)

## Contexto

Um mercadinho precisa de um sistema simples para registrar compras dos
clientes.

## Desafio

Desenvolva um programa que:

-   Armazene produtos e preços\
-   Permita ao cliente escolher produtos\
-   Calcule o valor total da compra\
-   Mostre um resumo final (tipo nota fiscal simples)

## Requisitos obrigatórios

-   Utilizar listas para armazenar produtos e preços\
-   Utilizar tuplas para representar cada produto `(nome, preço)`\
-   Utilizar estrutura de repetição (while) para continuar comprando\
-   Usar estrutura de decisão (if) para validar opções\
-   Criar variáveis para:
    -   Total da compra\
    -   Quantidade de itens

## Desafio extra

-   Aplicar desconto para compras acima de um valor\
-   Mostrar o produto mais caro da compra
